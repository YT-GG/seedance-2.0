## 📖 一句话总结

**Seedance 2.0 Skill OS** 是一个 AI 视频生成工具，它让你像导演一样控制视频生成，而不是盲目调参。核心理念：**描述你的意图，系统会处理细节**。

## 💡 5 分钟快速上手

### 场景 1：我有一个模糊的想法

**你想做什么：** 脑子里有个感觉，但说不清楚

**该用哪个 Skill：** `seedance-interview`

**对话示例：**
```
@seedance-20

我想要一个视频，有一个女人在房间里，感觉很孤独，但我不知道怎么拍才能表达这个感觉
```

**系统会做什么：**
- 问你几个关键问题（不会烦人，就3-5个）
- 帮你理清故事节奏、镜头感、光影效果
- 最后给你一份完整的"制片简报"
- 自动生成可以直接用的视频提示词

---

### 场景 2：我已经知道要拍什么

**你想做什么：** 清楚地描述一个场景

**该用哪个 Skill：** `seedance-prompt`

**对话示例：**
```
@seedance-20

场景：一个咖啡馆里，女主角收到一条信息，她表情从惊讶变成失望，
特写镜头，自然光从窗户射进来，咖啡杯在前景有点模糊
```

**系统会做什么：**
- 优化你的描述（去掉"美的"、"电影感"这样的虚词）
- 转成 Seedance 平台能理解的专业提示词
- 考虑镜头、灯光、声音、演员表现是否一致
- 输出可以直接发给 Seedance 的 Prompt

---

### 场景 3：我想做一个长故事（多个视频片段）

**你想做什么：** 一个故事分成 3-5 个视频片段

**该用哪个 Skill：** `seedance-sequence`

**对话示例：**
```
@seedance-20

我要做一个 30 秒的小故事：
- 第 1 段（10秒）：女主在地铁站等车
- 第 2 段（10秒）：她上车，见到一个人
- 第 3 段（10秒）：他们有眼神接触，镜头拉开

请帮我把每段都规划好
```

**系统会做什么：**
- 整理故事节奏和连接点
- 为每一段都生成专业 Prompt
- 记录每段的"起点"和"终点"（下一段怎么接）
- 确保前后视觉风格一致

---

### 场景 4：上一段视频生成好了，我要继续下一段

**你想做什么：** 基于已经生成的视频继续

**该用哪个 Skill：** `seedance-continuation`

**对话示例：**
```
@seedance-20

上一段的视频完成了（链接：xxx）
我要继续下一段。前面结束时，主角正在走向窗边
现在需要他转身，看向外面的雨景，特别无助
```

**系统会做什么：**
- 分析你上一段的**实际画面**（不是假设）
- 找到正确的"接口点"
- 生成下一段 Prompt，确保画面连接流畅

## 🎬 核心概念速成

### 💬 问你自己这 4 个问题

在写任何 Prompt 前，先问自己：

1. **这是什么戏？** 
   - 一个人物的感受、一个转折、一个展示？

2. **怎么用镜头表达它？**
   - 远景显示孤独、特写看表情、摇镜头制造紧张感？

3. **灯光怎么帮我？**
   - 冷色调表达悲伤、暖色调表达温暖、硬光制造冲突？

4. **声音呢？**
   - 安静突出内心、音乐推动节奏、脚步声表达焦虑？

### ✨ 一个例子对比

**不推荐（虚浮）：**
```
一个女人在咖啡馆，感觉很美，电影感很强，4K，cinematic
```

**推荐（有视觉）：**
```
中景，女人坐在咖啡馆窗边，侧光勾勒脸部轮廓，
她缓缓转身看向窗外的雨，没有音乐，只有雨声和咖啡杯碰碟的声音
```

## 🎯 使用流程总结

### 最简单的流程（1 个视频片段）

```
"我想要一个视频..."
    ↓
  seedance-interview （如果想法不清楚）
    ↓
  seedance-prompt （生成提示词）
    ↓
  发送到 Seedance 平台
    ↓
  得到视频 ✅
```

### 稍微复杂的流程（3 个连接片段）

```
"我要做一个故事..."
    ↓
  seedance-sequence （规划全部片段）
    ↓
  生成 Clip 1, 2, 3 的提示词
    ↓
  发送 Clip 1 到 Seedance
    ↓
  视频返回，用 seedance-continuation 继续 Clip 2
    ↓
  收到 Clip 2，继续 Clip 3
    ↓
  所有片段连接成完整视频 ✅
```

**用户输入：**
```
@seedance-20

我想做一个视频，场景是一个女孩独自在海边散步，
夕阳西下，她表情从忧伤逐渐变得释然。
我想要一种治愈的、温暖的感觉，但不是唯美的滤镜那种，
```

**系统会问你：**
1. 视频长度？（5 秒还是 15 秒？）
2. 夕阳是主角吗？（镜头多久看夕阳，多久看她？）
3. 这是什么故事的转折点吗？（她在想��么？）
4. 声音呢？（只有海浪声？还要背景音乐？）
5. 最后一秒她的表情定格什么？（笑？平和？希望？）

**系统给你生成的提示词示例：**
```
女孩单独在沙滩上缓缓漫步，夕阳投射长影，逆光勾勒她的轮廓。
镜头在她身后，缓慢跟随，保持低角度看她的剪影。
她起初面无表情望向远方，5秒后逐渐抬起视线，嘴角浮起淡淡微笑。
金色的夕光柔和地照亮她的脸侧面，海浪声伴随她的呼吸。
最后一秒，她停下脚步，转向镜头，眼中有光。
```

### 例 2：多段故事

**用户输入：**
```
@seedance-20

我要做一个 30 秒的短剧：
开场：一个人在地铁站台等车，焦虑感很强
第二段：车来了，他上车，看到对面坐着一个认识的人
结尾：他们相视一笑，陌生感消散

这个故事想表达：在城市的孤独中忽然遇见熟悉的人
```

**系统会：**
1. 把故事分成 3 个 10 秒的片段
2. 为每段确定"起点"和"终点"（镜头位置、人物位置、表情等）
3. 逐段生成 Prompt

**生成的第一段 Prompt：**
```
地铁站台，黄色荧光灯，人来人往但有点虚化。
镜头对着一个男人，他穿着商务装，不断看手机又看屏幕上的时间。
肩膀有点紧张，眼神飘忽，一次又一次抬起头看前方。
背景音：站台广播声、脚步声，制造一种城市的疏离感。
10 秒内，远处传来地铁车轮声，他的身体稍稍前倾，等待。
```

**生成的第二段 Prompt：**
```
基于上一段：他已经听到地铁声，现在镜头切到——
车门打开，人群涌入。
他随着人流上车，镜头跟随他找座位。
他坐下后，抬头看对面，忽然愣住。
（这里会精确记录：他坐的位置、抬头的角度、转身的速度）
```

**生成的第三段 Prompt：**
```
他和对面的人四目相对，大概 2 秒的沉默。
然后两人同时露出笑容，一种"真的是你"的惊喜感。
镜头拉开一点点，看两个人在地铁车厢里的这一刻亲密但又公众的感觉。
列车平稳行驶，光影掠过他们的脸。
最后 1 秒是一个安静的、满足的定格。
```

## ⚠️ 新手常见误区

### ❌ 错误 1：堆砌"美"的词

```
错：epic cinematic masterpiece ultra 4K stunning beautiful amazing
对：远景低角度，女人走过金色麦田，背光，每一根麦子都有光
```

### ❌ 错误 2：忘记声音

```
错：一个人在房间里，表情很悲伤
对：一个���在房间里，背景是冰箱的嗡嗡声和窗外的雨声，她无声地哭
```

### ❌ 错误 3：混乱的多人场景

```
错：三个人在一起，各种表情和动作
对：第一个人说话，第二个人听，第三个人在背景
（多人的故事分成多个短视频会更清晰）
```

### ❌ 错误 4：不确认长度和平台

```
错：直接给一个 40 秒的故事要求
对：告诉系统"我要在 TikTok/Instagram 用，竖屏 9 秒"
```

### ❌ 错误 5：一个 Prompt 里写 5 个镜头的电影

```
错：开场远景，然后特写，然后移动镜头，然后...
对：一个 Prompt = 一个镜头角度 = 一个视频片段
```

## 🔑 关键概念

### 什么是"提示词"（Prompt）？

你给 Seedance 的**自然语言指令**，告诉它：
- 什么人物、什么场景
- 镜头怎么摆
- 灯光什么效果
- 声音有什么
- 人物怎么动

**示例：**
```
中景，一个穿黑色毛衣的女性���在窗边，窗外是下午的阳光。
她低头看手机，突然皱起眉头，把手机放下。
镜头缓缓推进，她的侧脸进入特写，一滴眼泪滑过脸颊。
背景是窗外树叶的影子晃动，音乐很安静，只有呼吸声。
```

### 什么是"序列"（Sequence）？

多个**连接的视频片段**组成一个完整故事。
- 每段 5-15 秒
- 片段之间要有逻辑连接点
- 前一段的"结尾"决定下一段的"开头"

### 什么是"参考资产"（Reference Asset）？

你提供给 Seedance 的**参考材料**：
- **参考图片**：一个人物的形象、场景的风格
- **参考视频**：一个运动方式、镜头节奏
- **参考音频**：对白、音乐、环境音

每个参考有特定的**用途**（角色用这个、光线用那个）。

## 🚨 重要限制

### ✋ Seedance 不能做这些

1. **生成人脸**：不能凭空创造真实人物（用真人演员或用参考图片）
2. **直接使用名人形象**：不能直接用明星，要用 `seedance-copyright` 转换
3. **超长视频一次生成**：单次最多生成 10-15 秒（更长要分段）
4. **完美的多人对话**：多人同时说话容易乱，分成单人 + 交互
5. **实时渲染**：不能用作直播，只能生成后再用

### ✅ Seedance 擅长做这些

1. **单人故事**：一个人的表情变化、动作
2. **环境和运动**：天气、物体移动、自然风景
3. **镜头感**：推镜、拉镜、平移，光影变化
4. **短剧表现力**：5-15 秒的情感转折
5. **产品展示**：商品在各种背景/光线下的呈现
